🌾 Agricultural Yield Prediction Dashboard

This project presents an interactive Power BI dashboard designed to analyze and visualize crop yield data across multiple regions, crops, soil types, and environmental conditions. Built using a synthetic dataset of 1 million agricultural records, the dashboard enables users to explore actual vs predicted yields, understand key factors influencing productivity (like rainfall, temperature, irrigation, and fertilizer usage), and dive deep into regional and crop-specific performance.
